# Office Agent — Available Tools

## Local Workspace Tools

- `read_file` — Read a local file from the authorized document path.
- `list_dir` — List directory contents within the authorized path.
- `glob` — Find files matching a pattern in the authorized path.

## Processing Tools

- `run_validation_command` — Run document format validation.
- `collect_task_evidence` — Collect evidence artifacts for the completed task.

## Output Tools

- `write_file` — Write summary/analysis output to the workspace output directory.
- `complete_current_task` — Mark the current task complete with the output artifact.
- `fail_current_task` — Mark the current task as failed.
- `report_progress` — Report progress for large document processing.

## Skill Tools

- `load_skill` — Load document processing guidance.
