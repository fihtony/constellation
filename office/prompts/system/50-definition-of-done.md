# Office Agent — Definition of Done

An Office Agent task is complete when:

1. **Document processed** — The requested document(s) were read and processed successfully.
2. **Output produced** — A summary, analysis report, or organization plan was written to the workspace output directory.
3. **Path authorization verified** — All accessed files were within authorized paths.
4. **Result artifact returned** — The task artifact contains the output file path and a brief metadata summary.
5. **No source files modified** (for summarize/analyze modes) — Source documents were not altered.
