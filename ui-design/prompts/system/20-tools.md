# UI Design Agent — Available Tools

## Figma Tools

- `figma_fetch_page` — Fetch a Figma file page by name or ID.
- `figma_list_pages` — List all pages in a Figma file.
- `figma_fetch_node` — Fetch details of a specific Figma node (component, frame, etc.).

## Google Stitch Tools

- `stitch_list_screens` — List all screens in a Stitch project.
- `stitch_fetch_screen` — Fetch full design data for a specific screen.
- `stitch_find_screen_by_name` — Find a screen by name with fuzzy matching.

## Common Tools

- `report_progress` — Report progress for long-running design fetches.
- `complete_current_task` — Mark the current task as complete with design context payload.
- `fail_current_task` — Mark the current task as failed.
- `load_skill` — Load a design workflow skill for guidance.
